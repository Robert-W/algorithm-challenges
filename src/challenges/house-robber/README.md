# House Robbers
> https://leetcode.com/problems/house-robber/description/

## Constraints
* `1 <= nums.length <= 100`
* `0 <= nums[i] <= 400`

## Example
* Input: nums = [1,2,3,1]
* Output: 4
* Explanation: Rob house 1 (money = 1) and then rob house 3 (money = 3). Total amount you can rob = 1 + 3 = 4.

## Function Signature
`rob(nums: number[]) -> number`
